# 📊 Customer Churn Prediction

## 📌 Overview

Customer churn is a critical problem for businesses as it directly impacts revenue and growth. This project uses machine learning techniques to predict whether a customer is likely to leave a service based on historical data.

---

## 🎯 Objective

The main goal of this project is to:

* Predict customer churn using machine learning models
* Identify key factors affecting customer retention
* Help businesses take proactive actions to reduce churn

---

## 🗂️ Dataset

This project uses multiple datasets including:

* Bank Customer Churn Dataset
* Credit Card Churn Dataset
* E-Commerce Customer Dataset
* Netflix User Dataset

Each dataset contains customer-related information such as demographics, usage behavior, and account details.

---

## ⚙️ Technologies Used

* Python 🐍
* Pandas & NumPy
* Matplotlib & Seaborn
* Scikit-learn
* Jupyter Notebook

---

## 🧠 Machine Learning Models

The following models are used:

* Logistic Regression
* Decision Tree
* Random Forest

These models are trained and evaluated to find the best performance for churn prediction.

---

## 🔄 Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation

---

## 📈 Results

* Achieved good accuracy in predicting customer churn
* Identified important features influencing churn
* Improved prediction performance using ensemble methods

## 🚀 How to Run the Project

1. Clone the repository:

```

```

2. Navigate to the project folder:

```
cd Customer_Churn_Prediction
```

3. Install dependencies:

```

```

4. Run Jupyter Notebook:

```
jupyter notebook
```
